Classic Snake Game
This is a classic Snake game built using the Pygame library in Python. The game features a border, a timer, and a high score system that persists across sessions.

How to Play
Controls
Arrow Keys: Move the snake (Up, Down, Left, Right).

P: Pause and unpause the game.

F: Toggle fullscreen mode.

C: Restart the game after a "Game Over" screen.

Q: Quit the game from the "Game Over" screen.

High Score
The high score is saved to a file named highscore.txt in the same directory as the script, so your highest score is remembered for your next game.

Features
Resizable Window: The game window can be resized, and the game will adapt its border and playing area.

Difficulty Selection: Choose between Easy, Medium, and Hard speeds at the start menu.

Timer: Tracks your elapsed time during gameplay.

Dependencies
You need to have the Pygame library installed to run this game. You can install it using pip:

pip install pygame

How to Run
Make sure you have the required dependencies installed.

Run the script from your terminal:

python snake.py
